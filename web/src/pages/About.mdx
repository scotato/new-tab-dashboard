# Sleeper

* <Emoji>😴</Emoji> An ambient new tab browser extension for <Link to={process.env.REACT_APP_CHROME_STORE}>Chrome</Link>, <Link to={process.env.REACT_APP_FIREFOX_STORE}>Firefox</Link> & <Link to={process.env.REACT_APP_MICROSOFT_STORE}>Edge</Link>

Sleeper is an extension for desktop browsers that replaces your new tab screen with the sleeper app. Sleeper is an open source project available on <Link to={process.env.REACT_APP_GITHUB_REPO}>GitHub</Link>, where you can file issues and monitor development progress.

## Features

* <Emoji>⭐️</Emoji> Top sites
* <Emoji>🧙‍♀️</Emoji> Automatic dark mode
* <Emoji>🛌</Emoji> Blobs wake up every minute
* <Emoji>🐋</Emoji> Blobs of greater size have more mass, affecting their movement
* <Emoji>😲</Emoji> Click a blob to wake it up
